# ror-react-mongo
Example repository to Getting started with RoR, React &amp; MongoDB

## Development :rocket:

### server
 - `cd server`
 - Install the gems using `bundle install`
 - Change DB details of mongo in `config/mongoid.yml`
 - Start server using `rails s`

### client

 - `cd client`
 - `yarn install`
 - `yarn start`
